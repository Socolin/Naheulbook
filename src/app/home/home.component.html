Naheulbook

Naheulbook est un site/application gratuit permettant de gérer les fiches de personnages du jeu de rôle Naheulbeuk simplement.

Cette application n'a pas pour but d'automatiser le jeu, ou de faire un équivalent de Roll20, mais juste de simplifier la gestion de certains mécanismes complexes ou qui prenne du temps, et de pouvoir plus ce concentrer sur le roleplay.

Naheulbook est utilisable sur smartphone pour le mode joueur, le mode MJ est pensé pour fonctionner sur une tablette (ou écran plus grand)

L'application est encore en cours de développement et il manque encore certaines choses. Le code source est disponible ainsi que le gestionnaire de ticket afin de suivre les fonctionnalités en cours de développement.
