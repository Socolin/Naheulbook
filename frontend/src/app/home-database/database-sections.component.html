Cette section permet d'accéder à un certains nombres d'informations utiles pour les Maître de jeu.

Si vous êtes un joueur vous ne devriez pas avoir besoin de venir ici.

Si vous êtes un Maître de jeu vous pouvez activer le mode Maître de jeu dans le menu accessible via le bouton menu_vert en haut à gauche de cette page, afin d'avoir accès à toutes les informations.
